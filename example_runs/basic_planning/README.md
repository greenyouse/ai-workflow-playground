# Basic Planning

This is the first run through the planner without an optimizations.

It's just pairs custom agents with custom tasks.

Try running this with:
```bash
source .venv/bin/activate
run_crew planning {idea}

```

Example project idea prompt I used:

```text
I want to set up a personal blog using Hugo.

The blog should be fast, simple, and easy to maintain long-term. I plan to write about software engineering, side projects, and learning notes.

I’d like it to be deployed somewhere cheap or free, ideally with a simple CI/CD setup.

I also want basic features like tags, categories, RSS feed, and a clean design.

I’m not sure which theme to use, how to structure content, or how to handle things like SEO, analytics, and future customization.

It would be nice if I could extend it later with things like search or comments, but that’s not required initially.
```
